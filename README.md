# Archery Note 🏹

アーチェリーの得点記録＆サイト調整サポートアプリ（スマホ対応・単一HTML）。

## 使い方

ローカルで開くだけでも動きます（`index.html` をブラウザで開く）。
スマホで使うには、どこかに置いて（おすすめは GitHub Pages）URLを開き、
ブラウザのメニューから **「ホーム画面に追加」** するとアプリのように使えます。
HTTPSで開けばオフラインでも動作します（Service Worker対応）。

ローカルでの動作確認用サーバー:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File serve.ps1
# → http://localhost:8741/
```

## 機能

- **記録**: 的面をタップ＆ドラッグで着弾位置を入力（押したまま0.4秒で微調整モード、拡大レンズ付き）。点数は座標から自動採点（X/10〜1/M）。エンド制で集計。距離は70/50/30/18m＋自由設定、的は122/80/60/40cm。
- **履歴**: 練習一覧・着弾プロット・エンド表・グルーピング中心/半径(RMS)・距離別平均点の推移グラフ。
- **サイト台帳**: セッティング×距離ごとにサイト値（上下/左右）を日付つきで台帳管理。練習終了時にワンタップで台帳へ記録。
- **調整アドバイス**:
  - 最新練習のグルーピング中心から「サイトをどちらへ・何mm（・何クリック）」を提案
  - サイト値を数値で記録した複数回の練習から、回帰分析で「ズレが0になるサイト値」を推定
  - mm目安はアイ〜サイト距離（設定可、既定850mm）から幾何計算。クリック換算は「1クリック=◯cm@70m」を一度測って登録すると有効（他距離へ自動換算）
- **用具**: 弓・リム・ポンド・矢・ブレースハイト等のセッティング管理と変更履歴の自動記録。
- **データ**: 端末内（localStorage）保存。JSONでバックアップ/復元可能。

## ファイル

- `index.html` — アプリ本体（依存ライブラリなし）
- `manifest.json` / `sw.js` / `icon.svg` — PWA用（ホーム画面追加・オフライン動作）
- `serve.ps1` — ローカル確認用の簡易サーバー
