# Demo Script (under 3 minutes)

- **0:00–0:15 — Problem:** “Scores tell me what happened, but not what to try next.” Show the empty first-run screen.
- **0:15–0:30 — App:** Choose “Try fictional demo data.” State that records stay on this device.
- **0:30–0:50 — Record:** Open Record and place an arrow on the target; point out the fast phone-first flow.
- **0:50–1:15 — Dashboard:** Open Analysis. Read last practice, weekly sessions/arrows, recent average and previous-practice delta.
- **1:15–1:35 — Grouping:** Show center, horizontal/vertical spread, RMS radius and outlier explanation.
- **1:35–1:55 — Timeline:** Switch 7, 30, 90 days and all time. Point out text labels and chart title.
- **1:55–2:15 — Next practice:** Read one suggestion and its recorded reason; explain that rules are local and reproducible.
- **2:15–2:30 — Offline:** Enable offline mode, reload, and return to Analysis.
- **2:30–2:45 — Codex:** Show tests and this development log; explain the inspect, test, implement, verify loop.
- **2:45–2:55 — Close:** “Archery Note turns private records into the next practice action.”
