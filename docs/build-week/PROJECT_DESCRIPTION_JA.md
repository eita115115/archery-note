# Archery Note

**一文:** アーチェリーの記録を、根拠の分かる次の練習行動へ変える、プライベートなオフライン練習コーチです。

対象は、通信が不安定な射場でスマートフォンを使う大学アーチェリー部員と初心者です。得点、着弾、用具、サイト、射形の記録を端末内で組み合わせ、成長ダッシュボード、時系列比較、ロバストなグルーピング分析、次回試すことを表示します。

単なる得点台帳と異なり、他人の絶対値ではなく自分の過去記録と比較します。提案には必ず記録上の理由を添え、記録が少ないときは信頼度を下げます。外部サーバー、分析SDK、アカウントは使いません。

構成は Vanilla JavaScript、HTML/CSS、端末内ストレージとスナップショット、JSON/CSV、Service Worker、Node検査、Playwrightです。Codexはリポジトリ調査、純関数設計、TDD、画面統合、ブラウザ検証、セキュリティ境界の確認、応募資料作成に使用しました。Build Weekでは7/30/90日表示、成長ダッシュボード、根拠付き提案、分離して削除できる架空デモデータを追加しました。
