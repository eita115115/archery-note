# Code of Conduct

## Our Pledge

Archery Noteのコミュニティでは、経験、立場、利用環境に関係なく、参加者が安心して協力できる場を目指します。

## Expected Behavior

- 相手を尊重し、具体的で建設的に話す。
- 不具合報告では、再現手順と環境をできるだけ明確に書く。
- 初心者や競技経験の違いを理由に相手を軽んじない。
- 安全性、アクセシビリティ、データ保護に関わる懸念を歓迎する。

## Unacceptable Behavior

- 攻撃、嫌がらせ、差別的な発言。
- 個人情報や練習データの無断公開。
- 公開Issueでの脆弱性詳細の開示。
- 継続的な議論妨害や悪意あるPR。

## Enforcement

問題がある場合は、公開Issueで争わず、メンテナーへ非公開で連絡してください。セキュリティに関わる内容は `SECURITY.md` の報告方法を使ってください。

メンテナーは、必要に応じてコメントの非表示、Issue/PRのロック、参加制限などを行います。
